# NHL Player Trade Target Analysis

This project is a Python-based exploratory data analysis (EDA) notebook that compares **Cale Makar** and **Quinn Hughes** as offensive trade targets.

The goal of the project was to answer one main question:

**If a general manager could trade for only one elite defenseman, which player would be the better offensive choice?**

---

## Project Overview

In this notebook, I used NHL player performance data to compare the offensive impact of Cale Makar and Quinn Hughes across their shared seasons from **2019 to 2024**.

The analysis focused on offensive metrics such as:

- Goals
- Primary Assists
- Points
- Shots on Goal
- Expected Goals (xG)

I followed a step-by-step EDA process:

1. Ask questions
2. Load and prepare the data
3. Explore the data with charts and summary tables
4. Draw insights and make a final recommendation

---

## Tools Used

- **Python**
- **Pandas**
- **Matplotlib**
- **Jupyter Notebook**

---

## Key Questions

This project was guided by the following questions:

- Which player produced more offense from season to season?
- Who had the stronger overall totals across the shared sample?
- Who was more efficient on a per-game basis?
- Did either player score above or below expected goals?
- Which player looks like the better offensive trade target based on the data?

---

## Data Preparation

To make the comparison fair and focused, I:

- Filtered the dataset to include only **Cale Makar** and **Quinn Hughes**
- Kept only the **"all" situation**
- Selected the offensive columns most relevant to the assignment
- Removed Quinn Hughes’s small 2018 sample from the main comparison
- Created per-game metrics such as:
  - Goals per game
  - Primary assists per game
  - Points per game
  - Shots per game
  - xGoals per game

---

## What I Analyzed

The notebook includes:

- Season-by-season points trends
- Goals vs. primary assists comparison
- Six-season total offensive production
- Per-game offensive efficiency
- Goals vs. expected goals (xG)
- Consistency using points-per-game averages and variation

---

## Main Findings

Some of the main findings from the analysis were:

- **Cale Makar** had more total points across the shared sample
- Makar had a much stronger goal-scoring profile
- Makar generated more shots on goal and higher expected goals
- **Quinn Hughes** stood out as a strong playmaker, especially in primary assists
- On a per-game basis, Makar was stronger in several major offensive categories
- Makar also looked slightly more consistent at an elite offensive level

---

## Final Recommendation

Based on the offensive metrics used in this project, I recommended **Cale Makar** as the better offensive trade target.

While Quinn Hughes performed very well as a playmaker and remained an elite offensive defenseman, Makar showed the stronger overall offensive profile across the 2019–2024 shared sample.
